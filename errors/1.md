# Graph has no nodes

The error message looks something like this:
```
Runtime error: "[00:00:02:0167]:[MapLoader]: Graph has no nodes for some reason." at line 385, column 5 in file "Includes\WaspLib\main\position\map\maploader.simba"
Stack Trace:
  0) Line 385 in function "TRSMapLoader.CacheGraph" in file "Includes\WaspLib\main\position\map\maploader.simba"
  1) Line 470 in function "TRSMapLoader.GetGraph" in file "Includes\WaspLib\main\position\map\maploader.simba"
  2) Line 483 in function "TRSMapLoader.SetupGraph" in file "Includes\WaspLib\main\position\map\maploader.simba"
  3) Line 561 in function "TRSMapLoader.Add" in file "Includes\WaspLib\main\position\map\maploader.simba"
  4) Line 690 in function "TRSMapLoader.Load" in file "Includes\WaspLib\main\position\map\maploader.simba"
  5) Line 157 in function "TRSMap.Setup" in file "Includes\WaspLib\main\position\map\map.simba" 
```

The solution is to delete the `Simba/Data/Cache/` directory which you can do easily through the wasp-launcher settings.

If you continue having this issue, it's recommended you delete `Simba/Data/Cache/` again and also delete `Simba/Data/Assets/` which can also be deleted from the wasp-launcher settings.

This will force WaspLib to redownload fresh assets for the scripts.

If after this the issue persists, it means the issue is related to the script or WaspLib in someway, please report the bug in [🙋help](https://discord.com/channels/1404396749350506496/1411998010975453285)
