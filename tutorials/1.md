In this guide I want to give a quick run-down how to setup profiles to hop to user defined worlds.

**Step One**
Start a script and you should be greeted with a GUI. If no GUI is present, then the script won't support this feature.

**Step Two**
Goto the bottom of the GUI and find the button that says ``Edit Profile`` Click this.

**Step Three** 
Enter Bank Pin if you have one. Username and password can be entered if you using Runelite Legacy Launcher.

**Step Four**
You'll see checkboxes. Checking stuff will include them when you generate worlds. So choose all the filters that you would like to apply yourself.

**Step Five** 
After you have chosen all the filters you want click ``Generate Worlds``. You should see a new list in the display box with worlds you filtered.

**Step Six** 
Click Save Profile. Whatever worlds are currently showing in the display box are the worlds your profile will hop too.Congrats. You set up your first profile!
